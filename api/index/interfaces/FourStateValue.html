@celox-sim/celox / index / FourStateValue

Interface: FourStateValue ​

Defined in: packages/celox/src/types.ts:269

A 4-state value with explicit bit-level mask.

Encoding per bit:

  • (v=0, m=0) → 0
  • (v=1, m=0) → 1
  • (v=0, m=1) → Z (high-impedance)
  • (v=1, m=1) → X (unknown)
